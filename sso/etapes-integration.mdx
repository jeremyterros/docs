## Configuration technique

<Steps>
  <Step title="Comptes de test">
    Créer des **comptes de test** pour chaque service.
  </Step>
  <Step title="Configuration back-office">
    Compléter la configuration dans le back-office.
  </Step>
  <Step title="Import des métadonnées SAML">
    Importer les métadonnées SAML du fournisseur d'identité.
  </Step>
</Steps>

## Tests fonctionnels

Une fois la configuration technique terminée, les tests suivants doivent être effectués :

| Test | Description |
| --- | --- |
| **Connexion automatique SSO** | Vérifier que l'utilisateur est connecté automatiquement via le SSO |
| **Rechargement de badge** | Tester avec un montant réel et un montant de 0 |
| **Webview** | Vérifier l'ouverture transparente sans connexion manuelle |
| **Click & Collect** | Simuler une commande et un paiement |
