L'utilisateur peut accéder à son profil pour consulter toutes les informations liées à son compte.

## Informations disponibles

| Champ | Modifiable |
| --- | --- |
| Restaurant | Non |
| Email | Non |
| Prénom | Non |
| Nom | Non |
| Numéro de badge | Non |
| Numéro de téléphone | **Oui** |
| Mot de passe | **Oui** |
| Notifications ON/OFF | **Oui** |

## Actions disponibles

- **Modifier le numéro de téléphone** : l'utilisateur peut mettre à jour son numéro de téléphone directement depuis son profil.
- **Modifier le mot de passe** : l'utilisateur peut changer son mot de passe à tout moment.
- **Se déconnecter** : bouton de déconnexion accessible depuis le profil.
- **Supprimer son compte** : l'utilisateur peut demander la suppression de son compte.
