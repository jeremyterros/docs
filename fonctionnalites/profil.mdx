L'utilisateur peut acceder a son profil pour consulter toutes les informations liees a son compte.

## Informations disponibles

| Champ | Modifiable |
| --- | --- |
| Restaurant | Non |
| Email | Non |
| Prenom | Non |
| Nom | Non |
| Numero de badge | Non |
| Numero de telephone | **Oui** |
| Mot de passe | **Oui** |
| Notifications ON/OFF | **Oui** |

## Actions disponibles

- **Modifier le numero de telephone** : l'utilisateur peut mettre a jour son numero de telephone directement depuis son profil.
- **Modifier le mot de passe** : l'utilisateur peut changer son mot de passe a tout moment.
- **Se deconnecter** : bouton de deconnexion accessible depuis le profil.
- **Supprimer son compte** : l'utilisateur peut demander la suppression de son compte.
