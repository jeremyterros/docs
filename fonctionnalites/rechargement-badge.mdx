Il est possible de recharger son badge depuis le profil ou directement depuis le bouton **"Recharger mon badge"** accessible sur la page d'accueil.

## Points d'acces

- **Page d'accueil** : bouton "Recharger mon badge" visible directement.
- **Profil utilisateur** : option de rechargement accessible depuis la page profil.

## Fournisseurs

Deux options de fournisseur sont disponibles pour le rechargement :

- **Adoria**
- **Pi Electronique (PiPay)**

La configuration du fournisseur s'effectue via le back-office Serenest. Voir la section [Configuration du back-office SSO](/sso/configuration-back-office) pour plus de details.
