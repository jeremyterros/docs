L'utilisateur peut consulter les menus du restaurant en selectionnant le jour souhaite.

## Fonctionnalites

- **Selection par jour** : navigation entre les differents jours pour consulter les menus disponibles.
- **Detail des plats** : affichage des plats proposes pour chaque service (dejeuner, diner, etc.).
- **Informations nutritionnelles** : selon la configuration du site, des informations complementaires peuvent etre affichees sur les plats.
