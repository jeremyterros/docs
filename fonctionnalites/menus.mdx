L'utilisateur peut consulter les menus du restaurant en sélectionnant le jour souhaité.

## Fonctionnalités

- **Sélection par jour** : navigation entre les différents jours pour consulter les menus disponibles.
- **Détail des plats** : affichage des plats proposés pour chaque service (déjeuner, dîner, etc.).
- **Informations nutritionnelles** : selon la configuration du site, des informations complémentaires peuvent être affichées sur les plats.
